ADAMによる学習の可視化

         
直感でわかる最適化：SGD / Adam / AdamW（完全オフライン）
       
     
     
                     
   
 
 
   
           
       
         

モデル（何をシミュレート？）

         

            原点 (0,0) で最小になる 損失地形（細長い谷）を、ロボット θ=(x,y) が下っていく様子です。右の図は x–y 平面、赤がゴール。          

         
           
             
損失（谷の形）
                                                              L(x,y)                   =                   12                   (                   u2a2                   +                   v2b2                   )                                            
（MathML が未対応なら）L(x,y)=1/2*(u^2/a^2 + v^2/b^2)
           
           
             
座標の回転（谷の向き φ）
                                                [u,v]^T                   =                   R(φ)                   [x,y]^T                                            
（未対応なら）[u,v]^T = R(φ)[x,y]^T
           
         
         
               
  • θ=(x,y) はロボの位置（最小化したいパラメータ）。
    •            
  • a,b は谷の細さ（b≫a で横に長い）。
    •            
  • φ は谷の向き（回転角）。
    •          
         
            更新式（SGD / Adam / AdamW） — 詳細を見る            
             
SGD:  θ(t+1) = θ(t) − η ∇L(θ(t))
             
Adam: m_t=β₁ m_{t-1}+(1−β₁)g_t,  v_t=β₂ v_{t-1}+(1−β₂)g_t²
             
       θ(t+1)=θ(t) − η ·  m̂_t / (√v̂_t + ε)
             
AdamW: θ(t+1)=θ(t) − η · ( m̂_t/(√v̂_t+ε) + λ θ(t) )

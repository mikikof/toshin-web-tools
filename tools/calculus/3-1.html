円錐曲線ファミリー

       

            円錐曲線（conic section）とは、円錐を平面でまっすぐ切った時の切り口の形のことです。
            切る角度によって、おなじみの「円」や「楕円」から、「放物線」、「双曲線」まで、様々な曲線が現れます。        

       
           
               

円錐曲線の一般式 (2次式)

               
                   

Ax² + Bxy + Cy² + Dx + Ey + F = 0

               
               

※ A, B, C のうち少なくとも1つは0ではない

               

図形の種類と判別式

               
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               
図形の種類判別式 Δ = B² - 4AC補足条件
円Δ < 0B = 0 かつ A = C
楕円Δ < 0B ≠ 0 または A ≠ C
放物線Δ = 0-
双曲線Δ > 0-

ランダウ記号 o(x) をインタラクティブに学ぶ

           

近似計算の「誤差」を評価

       
               
           

1. なぜ「誤差の評価」が必要なのか？

           

数学や物理では、複雑な関数をシンプルな多項式で近似することがよくあります。例えば、sin(x) は x が0に近いとき、x でとてもよく近似できます。

           
                sin(x) ≈ x            
           

しかし、この近似には必ず「誤差」が伴います。その誤差はどれくらい小さいのでしょうか？「すごく小さい」だけでは不十分です。誤差がどれくらいの速さで0に近づくかを数学的に表現するために、ランダウの記号が使われます。

       
               
           

2. 「はるかに小さい」を体験する

           

ランダウ記号 f(x) = o(g(x)) は、「x がある値に近づくとき、f(x) は g(x) よりも圧倒的に速く0に近づく」ことを意味します。ここでは f(x) = x³ と g(x) = x² で、x → 0 の場合を体験してみましょう。

                    f(x)=x³:                    
               
           
           
               

結論: x が0に近づくにつれて、比率 f(x)/g(x) も0に近づいていきますね！これは数学的に lim(x→0) f(x)/g(x) = 0 と書け、まさに x³ = o(x²) が成り立つことを意味します。

           
       
               
           

3. テイラー展開で見る近似の精度

           

sin(x) のテイラー展開は、近似の次数を上げるほど精度が向上する様子を視覚的に示してくれます。下のボタンで近似式を切り替えて、グラフがどのように sin(x) の曲線に近づいていくか確認しましょう。

                       
                                                           
            

1次近似式 sin(x) = x + o(x) の o(x) は、誤差が x よりも速く0に収束することを保証しています。

       
               
           

4. 実用例：単振り子の近似

           

物理学の単振り子の例です。sin(θ) = θ + o(θ) なので、角度 θ が小さいとき sin(θ) ≈ θ と近似します。この近似がどれほど有効か、下のシミュレーションで確かめましょう。実線の振り子（正確な計算）と、半透明のゴースト（近似計算）のズレに注目してください。

ε-δ論法

           

直線ver.と曲線ver.

       
               
           

極限の定義 (ε-δ論法)

           

                ∀ε > 0,                 ∃δ > 0                 s.t.                 0 < |x-a| < δ                 ⇒                 |f(x)-P| < ε            

           

                このとき limx→a f(x) = P となる。            

       
       
                       
               

直線: f(x) = 2x + 1

               

x → 1 のとき P → 3

               
                                   
               
                   
                                                                   
                   
                                           
                   
                       

「δをεに連動」をオンにしてください。

                   
               
           
                       
               

曲線: f(x) = x²

               

x → 2 のとき P → 4

               
                                   
               
                   
                                                                   
                   
                                           
                   
                       

「δをεに連動」をオンにしてください。

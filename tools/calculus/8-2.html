1. 連続性の探求

                   

原点 (0,0) に近づく道筋を変えると、関数の値はどうなるでしょうか？ 直線 y=mx に沿って原点に近づいてみましょう。

                   
                                                                   
                   

                        y=mx に沿った極限値:                                            

                   

                        傾き m を変えると、極限値が変化することがわかります。近づき方によって値が異なるため、この関数は原点で連続ではありません。                    

               
                               
                   

2. 偏微分可能性の探求

                   

では、なぜ偏微分は可能なのでしょうか？ x軸、y軸に沿った断面（スライス）を見てみましょう。

                   
                                                                   
                                                            
                       

                                           
                   

                        x軸に沿う (y=0) と、常に f(x,0)=0 です。同様にy軸に沿う (x=0) と、常に f(0,y)=0 です。                         2Dグラフが示す通り、これらの軸上では関数は平坦であり、原点での接線の傾きは 0 です。                    

                   

                                                                                                            f                               x                                                         (                             0                             ,                             0                             )                             =                             0                                                     ,                                                                                                               f                               y                                                         (                             0                             ,                             0                             )                             =                             0

インタラクティブに広義積分を考える

           

“はみ出した” 積分の取り扱いを、動くグラフで直感的に理解する

       
       
                       
               

広義積分の2つのパターン

               
                   

動機とイメージ

                   

広義積分が扱う「普通の積分では計算できない面積」には、大きく分けて2つのパターンがあります。

                   

パターン1：積分区間は有限だが、関数が無限大になる

                   

区間の中で、グラフがどこかで無限大に吹き上がってしまう場合です。たとえば、下のグラフ を区間 (0, 1] で考えます。x=0 に近づくにつれて y は天井知らずに大きくなります。この「はみ出した」部分を扱うのが広義積分です。

                   

特異点 x=0 を直接は計算できないため、まず微小な区間 だけ避けて積分し、最後に の極限を取ります。この例では、面積はある有限の値 2 に収束します。

                   
                       

グラフ: (特異点あり)

                       
                       
                                                                                   
                                                                                                                           
                           
                                計算された面積:                                                                    ≈                                  1.414                            
                       
                   
                   
                   

パターン2：関数は有限だが、積分区間が無限に続く

                   

もう一つのパターンは、グラフが無限の彼方まで続いている場合です。例えば、関数 で、x=2 から無限遠点までの面積を考えてみましょう。

                   

この場合、まず有限な区間 [2, N] で面積を計算し、そのあとで上限 N を無限大に飛ばす極限を考えます。この例では、グラフの減衰が速いため、N を大きくしても面積は有限の値 0.5 に収束します。

                   
                       

グラフ: (無限区間)

                       
                       
                                                                                   
                                N → 2                                 N = 2                                 N → ∞                            
                           
                                計算された面積:                                                                    ≈                                  0.000                            
                       
                   
                   

このように「一見アウト」に見える2つのパターンを、極限操作で救済（あるいは発散することを確認）するのが広義積分の考え方です。

               
               
                   

定義

                   

区間 [a,b] 内に特異点 c がただ一つだけ存在し、それ以外では可積分な関数 f に対して、広義積分を次のように定義します。

                   
                                           
                   

この極限が存在するとき、「広義積分は収束する」と言います。

               
               
                   

解説

    “特異点”とは？
                                関数値が になったり、定義されない（分母が0になるなど）点を指します。

                           
    •                        
  •                            

    なぜ で切り取るのか？
                                問題となる特異点を直接扱うことはできません。そこで、まず特異点の周りに幅 の「立ち入り禁止区域」を設けて安全な範囲で積分します。その後、この禁止区域を限りなく狭く () していったときに、積分値がどうなるかを調べるのです。これが極限の考え方です。

                           
    •                        
  •                            

    複数の特異点がある場合
                                特異点が複数ある場合は、区間を特異点ごとに分割し、それぞれの区間で同様に極限を取ります。全ての部分積分が収束する場合に限り、全体の積分も収束します。

                           
    •

区間 [0, 1] 上の関数 を考えてみましょう。この関数は x=0.25 と x=0.75 の2箇所に特異点（グラフが無限大に発散する点）を持ちます。

                   

この場合、2つの特異点の周りにそれぞれ「立ち入り禁止区域」（赤色の部分）を設けます。そして、それ以外の安全な領域（青色の部分）で積分を計算し、最後に の極限を取ります。

典型例：べき型特異点

                   

区間 (0, 1] における関数 (s>0) を考えます。この積分が収束するかどうかは、指数 s の値に依存します。

                   
                                           
                   

下のグラフで指数 s を動かして、収束と発散の境界が s=1 であることを確かめてみましょう。面積が青色なら収束、赤色なら発散を示します。

                   
                       

グラフ:

区間が無限に延びる場合

               
                   

定義

                   

関数 f が [a, ∞) の任意の有界閉区間で可積分であるとき、無限区間での広義積分を次のように定義します。

                   
                   

この極限が存在すれば「収束する」と呼びます。(-∞, b] の場合も同様です。

               
               
                   

解説

                   
                           
  •                            

    端点が“見えない”ときの面積測定
                                これは、果てしなく長い廊下の床をワックスがけする場面に似ています。まず玄関から N メートル先までを塗り、その面積（必要なワックス量）を計算します。次に N をどんどん大きくしていったとき、必要なワックスの総量が有限の値に収まるかどうかを調べる、というイメージです。

                           
    •                        
  •                            

    左右とも無限大に延びる場合 (-∞ から ∞)
                                この場合は、途中の適当な点（例えば x=0）で区間を二つに分け、                            

                               

    として、二つの積分が両方とも収束することを確認する必要があります。「左右対称だから大丈夫」と安易に判断するのは危険です（例： は発散）。

                           
    •                    
                    

可視化：-∞ から ∞ への積分

                   

正規分布の確率密度関数でもおなじみのガウス関数 を例に、-∞ から ∞ までの積分を見てみましょう。この積分は、まず原点で2つに分割し、対称な区間 [-R, R] で積分したあと、 の極限を取ることで計算します。

                   
                   
                       

グラフ: (収束)

                       
                       
                                                                                   
                                R → 0                                 R = 2.00                                 R → ∞                            
                           
                                計算された面積:                                                                    ≈                                  1.764                            
                           

R を大きくすると、面積が に収束していく様子が分かります。

一方、見た目は似ていても、無限遠での減衰が遅いと積分は発散します。例えば、 という関数を見てみましょう。x が大きいとき、この関数は に近い振る舞いをします。このわずかな違いが、収束と発散を分けるのです。

                                計算された面積:                                                                    ≈                                  5.991                            
                       
                   
               
               
                   

典型例：べき関数

                   

区間 [1, ∞) における関数 を考えます。この場合も、収束性は指数 p の値に依存します。

                   
                   

ここでも p=1 が臨界点となります。有限区間の場合とは収束条件が逆になっている点に注目してください。面積が青色なら収束、赤色なら発散を示します。

絶対収束と条件収束

               
                   

定義

                   

広義積分 が収束し、さらに絶対値をとった積分 も収束するとき、f の積分は絶対収束すると言います。

                   

一方で、 は収束するが は発散するとき、この積分は条件収束すると言います。

                   

絶対収束は、数列の世界と同様に非常に強い収束性を示します。直感的には「プラスとマイナスの打ち消し合いに頼らずとも収束する」安定した状態です。
